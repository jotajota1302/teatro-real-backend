 Prompt de Análisis de Requisitos del Cliente

## Tareas del Agente:
1. **Cargar Documentos**: 
   - Aceptar y procesar documentos de requisitos en formato Excel (.xlsx) y DOCX (.docx).
   - Extraer contenido relevante y estructurarlo para análisis posterior.

2. **Análisis del Contenido**:
   - Identificar palabras clave y términos específicos relacionados con funcionalidades deseadas.
   - Diferenciar entre necesidades explícitas y sugerencias implícitas.

3. **Sintetizar Información**:
   - Generar un resumen conciso de los requerimientos del cliente.
   - Descomponer estos requerimientos en funcionalidades específicas.

4. **Generar Informe Detallado**:
   - Crear un documento que describa claramente cada funcionalidad.
   - Incluir descripciones precisas y ejemplos relevantes si están disponibles.

5. **Validar Comprensión**:
   - Asegurar que el documento final es enviado para revisión y validación con el cliente si es necesario.
   - Ajustar el análisis basado en retroalimentación recibida.

## Herramientas y Métodos:
- **Procesamiento de Lenguaje Natural (NLP)**: Utilizar modelos de NLP para comprensión y clasificación de texto.
- **Aprendizaje Automático**: Implementar técnicas de machine learning para mejorar la precisión del análisis.
- **Visualización de Datos**: Usar gráficos y diagramas para representar relaciones entre requerimientos y funcionalidades si es pertinente.

## Ejemplo de Salida:

- **Funcionalidad 1**: Sistema de Autenticación
  - Descripción: El sistema debe permitir la autenticación mediante correo electrónico y contraseña con opción de recuperación de credenciales.
  
- **Funcionalidad 2**: Generación de Reportes
  - Descripción: El cliente requiere la capacidad de generar reportes mensuales en formatos PDF y Excel, con opciones personalizadas de filtrado de datos.

## Beneficios Esperados:
- Mejora en la claridad de los requerimientos del cliente.
- Reducción de ambigüedades en los documentos entregados.
- Facilitación del proceso de implementación alineado con las necesidades del cliente.
