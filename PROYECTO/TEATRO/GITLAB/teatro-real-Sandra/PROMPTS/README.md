# Prompts Generados

Este directorio contiene los prompts específicos generados usando las templates del framework.

## Estructura

```
PROMPTS/
├── backend/          # Prompts para desarrollo backend
├── frontend/         # Prompts para desarrollo frontend
├── features/         # Prompts para features fullstack
└── bugfixes/         # Prompts para corrección de bugs
```

## Cómo Usar

1. Usa las templates de `../PROMPT_FRAMEWORK/` para crear nuevos prompts
2. Guarda el prompt generado en la carpeta correspondiente
3. Nombra el archivo de forma descriptiva: `[módulo]-[funcionalidad].md`

## Ejemplos de Nombres

- `backend/tempo-crud-espacios.md`
- `frontend/tops-editor-guion.md`
- `features/admin-gestion-usuarios.md`
- `bugfixes/tempo-calendario-no-carga.md`

## Templates Disponibles

Ver carpeta [PROMPT_FRAMEWORK](../PROMPT_FRAMEWORK/) para las templates base.
